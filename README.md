# multi-thread-crawle
A simple multi-threaded crawler for downloading high-definition wallpapers
这是我的第二个爬虫项目，相较于我的第一个项目，他已经不再那么的简陋，具有了多线程爬取和一定的反反爬能力。但是问题依旧明显，遇到反爬能力高的网站就显得力不从心。依次依然只推荐把该项目作为一个爬虫的素材来使用。
2026-6-15 简单的多线程爬虫
技术栈：urllib+requests+beautifulsoup4
挑战：1、单线程转变到多线程的过程中明显发现其中的知识更加的复杂且难以理解。
2、多线程下载中的多个任务爬取和多个页面爬取理解错误，导致误以为有几个线程只能下载几张图片，使得一开始的开发方向错误，为多个页面下载。
3、再开发了多线程下载的能力了之后，发现会下载所有源代码中携带”img"标签的图片有的甚至不是图片为垃圾文件，且无法指定下载我要的图片
4、因为单线程与多线程的代码整体结构有大量的不同，且缺乏开发项目的实战经验，导致开发初期有单线程转变为多线程的过程中明显的手足无措
